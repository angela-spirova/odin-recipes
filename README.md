# odin-recipes
[Project](https://www.theodinproject.com/lessons/foundations-recipes) for [The Odin Project](https://www.theodinproject.com/).

It's a simple recipe website consisting of a homepage and three recipe pages.

The project was created using HTML and its purpose is to showcase basic knowledge of the language, as well as of the command line and git.

Sources for the recipes used:
- [Brownies](https://www.allrecipes.com/recipe/25010/absolutely-best-brownies/)
- [Grilled cheese](https://www.allrecipes.com/recipe/125434/grilled-cheese-of-the-gods/)
- [Squash soup](https://www.allrecipes.com/recipe/77981/butternut-squash-soup-ii/)
